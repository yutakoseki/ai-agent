# services/agent-core

Python ベースの AgentCore 実行サービスを想定。

## 方針

- 受け取るリクエスト/ジョブは BFF で認証・RBAC・テナント終端済みのものに限定。
- 非同期キュー経由を優先し、同期 API は最小限。
- 入出力スキーマ検証・権限スコープチェック・リトライ/タイムアウト・監査ログを共通実装。

## 開発メモ

- 依存管理: Poetry/uv 想定
- デプロイ: コンテナ化してインフラ側で実行（Amplify とは別タスク/サービス）
- 環境変数: `config/env.example` に準拠し、必ずバリデーションを行う
