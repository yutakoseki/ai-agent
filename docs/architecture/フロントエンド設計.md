# フロントエンド設計

## カラートークン（Tailwind）

目的: `tailwind.config.cjs` のカラー定義を変更するだけで、UI全体の配色を切り替えられるようにする。

### デザイン方針
- 黒背景ベース + 鮮やかなアクセントを基本にする。
- フラットな見た目を優先し、影やグラデーションは最小限にする。

### 運用ルール
- 画面側での色の直書き（`#xxxxxx` や `rgba(...)`）は避け、`primary` / `secondary` / `accent` / `surface` などのトークンを使う。
- 背景グラデーションや影も `backgroundImage` / `boxShadow` にまとめ、クラス名で使う。

### 変更箇所
- `apps/web/tailwind.config.cjs`

### 例
```tsx
<div className="bg-surface text-primary">
  <button className="bg-primary text-secondary">
    ログイン
  </button>
</div>
```
